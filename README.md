# sol-e2e

E2E Testing of Solidity Contracts

## Nix

```bash
$ nix develop
(nix-shell-env) $ just
```

## Requirements

- [`rust`](https://rustup.rs)
- `forge` and `anvil` via
  [foundryup](https://book.getfoundry.sh/getting-started/installation)

### Tools

- [`just`](https://just.systems)
- [`cargo-nextest`](https://nexte.st)
